# Analisis-de-Datos-de-Taxis-en-Chicago
Proyecto del Sprint 8 del curso de TripleTen que combina consultas SQL y análisis en Python. Analiza datos de taxis en Chicago para identificar patrones, explorar los datos, crear visualizaciones y probar una hipótesis sobre la duración de los viajes en días lluviosos desde el Loop hasta el Aeropuerto O'Hare
